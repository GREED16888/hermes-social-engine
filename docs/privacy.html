Privacy Policy

Last updated: 2026-07-01

Hermes Social Engine is an owner-operated automation tool for publishing and managing posts on the owner's own social media accounts.

Information processed

  • OAuth tokens granted by the account owner for supported platforms.
  • Basic profile identifiers returned by platform APIs, such as account ID, display name, username, page ID, and page name.
  • Post content, captions, media file paths, upload status, provider post IDs, and release URLs.

How information is used

Information is used only to authenticate the owner, upload or schedule owner-approved content, verify upload status, prevent duplicate uploads, and maintain audit logs for the owner.

Data sharing

Hermes Social Engine does not sell data, does not use data for advertising profiles, and does not share data with third parties except the social platforms selected by the account owner for publishing.

Storage and security

Credentials and tokens are stored locally by the owner and are excluded from the public source repository. The public repository contains code and documentation only.

Data deletion

The owner may delete local tokens, logs, scheduled posts, and media references at any time from the local project directory. Revoking app access from the relevant platform account will also invalidate the platform token.

Contact

For questions, contact the repository owner through the GitHub repository associated with this project.
