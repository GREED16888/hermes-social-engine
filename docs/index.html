Hermes Social Engine

Hermes Social Engine is a private owner-operated tool for preparing, validating, scheduling, and uploading short-form videos and posts to the owner's own social media accounts.

Supported integrations

  • YouTube video upload via YouTube Data API
  • Facebook Page text and video publishing via Meta Graph API
  • TikTok OAuth and Content Posting API workflow for owner-approved uploads
  • Telegram notification and publishing workflow

TikTok usage

The TikTok integration is used only by the account owner to authenticate their own account, upload videos for review, and prevent duplicate or unsafe posting. It does not collect or sell user data.
